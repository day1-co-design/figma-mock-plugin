# [D1]목업 플러그인

Coloso Mock Data Plugin은 Figma 목업 제작에 필요한 크리에이터/강의 텍스트와 이미지를 선택한 레이어에 빠르게 넣어주는 플러그인입니다.

텍스트 레이어에는 크리에이터명, 직업명, 강의 제목 같은 문구를 랜덤으로 적용하고, 이미지 fill을 적용할 수 있는 레이어에는 프로필 이미지나 강의 썸네일을 랜덤으로 적용합니다.

## 주요 기능
- `프로필` 탭에서 아바타 이미지와 크리에이터 정보를 적용할 수 있습니다.
- `클래스` 탭에서 코스 이미지와 강의 제목을 적용할 수 있습니다.
- `전체`, `드로잉`, `영상 디자인`, `베이킹` 카테고리별로 데이터를 필터링할 수 있습니다.
- 여러 레이어를 선택하면 선택한 레이어마다 랜덤 데이터가 적용됩니다.
- 선택한 데이터 종류에 맞는 레이어가 없으면 하단 안내 문구와 비활성 버튼으로 상태를 알려줍니다.

## 사용법
처음 설치하는 디자이너는 [디자이너 설치 가이드](docs/designer-install-guide.md)를 먼저 확인하세요.

### 1. 레이어 선택
Figma 캔버스에서 데이터를 적용할 레이어를 먼저 선택합니다.

- 텍스트 데이터를 넣으려면 텍스트 레이어를 선택합니다.
- 이미지 데이터를 넣으려면 fill 적용이 가능한 프레임이나 도형 레이어를 선택합니다.

### 2. 탭 선택
플러그인 상단에서 적용할 데이터 맥락을 선택합니다.

- `프로필`: 크리에이터 프로필 관련 데이터
- `클래스`: 강의/코스 관련 데이터

### 3. 데이터 종류 선택
탭에 따라 사용할 데이터 종류를 선택합니다.

`프로필` 탭:
- `아바타`: 프로필 이미지 적용
- `크리에이터 정보`: 크리에이터명, 직업명 등 텍스트 적용

`클래스` 탭:
- `코스 이미지`: 강의 썸네일 이미지 적용
- `강의 정보`: 강의 제목 텍스트 적용

### 4. 데이터 유형 선택
필요한 경우 데이터 유형을 선택합니다.

크리에이터 정보:
- `크리에이터명`
- `직업 + 크리에이터명`
- `직업명`

코스 이미지:
- `코스 카드 썸네일`
- `상품 썸네일(1:1)`
- `상세페이지 커버 - PC`
- `상세페이지 커버 - Mobile`

강의 정보:
- `강의 제목`

### 5. 카테고리 선택
적용할 데이터 카테고리를 선택합니다.

- `전체`
- `드로잉`
- `영상 디자인`
- `베이킹`

`전체`를 선택하면 모든 카테고리 데이터가 랜덤 적용 대상에 포함됩니다.

### 6. 적용
하단 버튼을 눌러 선택한 레이어에 데이터를 적용합니다.

버튼 예시:
- `선택한 3 이미지 레이어에 적용`
- `선택한 2 텍스트 레이어에 적용`

선택한 레이어 타입이 현재 데이터 종류와 맞지 않으면 버튼이 비활성화되고 안내 문구가 표시됩니다.

## 데이터 소스
플러그인은 GitHub Pages에 배포된 JSON과 이미지 asset을 사용합니다.

- 텍스트 데이터: `data/text-course-info.json`
- 이미지 데이터: `data/image-assets.json`
- 이미지 파일: `assets/images/`

데이터를 수정한 뒤 저장소에 push하면 GitHub Pages 반영 후 플러그인에서 최신 데이터를 사용할 수 있습니다.
